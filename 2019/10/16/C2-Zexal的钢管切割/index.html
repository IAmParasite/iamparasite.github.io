C2-Zexal的钢管切割

Posted on 2019-10-16 | In 第二次上机题解 | | 阅读数

题面

在钢管切割的背景下，已经知道长度为1−n的钢管的价值，给定长度为n的钢管在切割若干次（也可以不切割）所带来的最小价值是？

输入

多组数据输入

第一行一个整数n，为起始钢管长度（0<n≤1000）

第二行n个整数，分别为长度为i的钢管的价值ti（0<ti≤1e6）

输出

对于每组数据，输出一行，为这根钢管所带来的最小价值T

输入样例

输出样例

分析题目

又是观众老爷们喜闻乐见的简单dp。

用p[i]来表示整个i作为一根钢管的价值

用f[i]表示长度为i分割后的最大值

有f[n] = min(f[n-i]+p[i], f[n])

可以知道的是，对于不同的分割方式，f[n]的值不同，那么就要遍历所有分割方式来确定f[n]
