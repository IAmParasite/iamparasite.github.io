C2-妙妙趣和值

Posted on 2019-10-18 | In 第二次上机题解 | | 阅读数

题面

定义一个长度为k的数组a1,a2,⋯,ak的妙妙趣值是min|ai−aj|(1≤i<j≤k)
那么请问对于一个长度为n的序列，所有长度为k的子序列的妙妙趣值和为多少。

某个序列的子序列是从最初序列通过去除某些元素但不破坏余下元素的相对位置（在前或在后）而形成的新序列。

答案为100000007取模

输入

第一行两个数n,k，表示序列的长度，子序列的长度。(2≤k≤n≤1000)

接下来一行，n个整数a1,a2,⋯,an,(0≤ai≤1e5)

输出

一行一个数，表示妙妙趣值和

输出样例

样例说明

长度为3的子序列有 [1,7,3], [1,3,5], [7,3,5], [1,7,5]，每一个妙妙趣值都是2

分析题目

我们最朴素的算法就是列举出所有长度为k的数列，再乘以他的妙妙趣和值，这种复杂度我们显然接受不了。

简单的思考以后，可以将数列从小到大排序，得到一个有序的序列。

我们尝试是否能算出对于一个固定的妙妙趣和值x的所有序列的个数呢？经过尝试我们很难统计这样的个数，在思考的过程中可以很容易确定妙妙趣和值大于等于x的序列个数，记为dp(x),我们可以一次自己计算出 ans += x * (dp(x) - dp(x-1))，容易想到这样是对的。这时x要从大往小枚举

我们再进行一些优化：

例如计算 1 * a1 + 2 * a2 + 3 * a3 + 4 * a4 的值，等同于 a1 + (a1 + a2) + (a1 + a2 + a3) + (a1 + a2 + a3 + a4)

这样我们可以从小往大遍历，dp(x) 从大往小一次减小，可以想到 x = 1 的情况为 a1 + a2 + a3 + a4，x = 2 的情况为 a1 + a2 + a3 以此类推

用dp[i][j]表示前i个数字中长度为k的序列，这个序列要满足的一定条件，即x+a[li]<=a[i]，li为满足不等式的最大值。递推方程式为dp[i][j] = dp[i-1][j] + dp[li][j]

递推方程式由两个部分由来：采取第k位和不采取第k位。不采取第k位显然满足条件，采取第i位要确定满足条件的下标。这样我们取得的dp[n][k]必然是满足妙妙趣和值大于x的数列个数
