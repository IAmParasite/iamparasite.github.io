C5-小面包

Posted on 2019-11-24 | In 第五次上机题解 |

题面

又要发小面包了。这次我们有许多3* 6的小面包和6* 6的方糕，以及一个6* N的长方形盒子，强迫症的某楠一定要把它们整齐的装到盒子里，并且要尽量装满。请问有多少总不同装法？

输入

对于每组数据，输出一行，为最终计算对1000007取模得到的结果。

输出

输出一行，能处理的垃圾最大重量，保留2位小数，单位为吨

输入样例

输出样例

Hint

输入为3时，只能放入一块小面包。

输入为6时，有三种情况：

（1）竖着放两块小面包

（2）横着放两块小面包

（3）放一块方糕

分析题目

这个题其实挺难想的，我想错了好久。

可以看到的是想要递推得到 f[i] 的答案，无非是从 f[i-1] 和 f[i-2] 递推而来，但是结尾不同可能会造成不同的答案。

比如结尾是 一个3*6的小面包的时候，想要递推得到下一个答案，可以将上一个小面包倒置过来横着放，又是一种不同的答案。

那么我们可以用f1[i] 表示 最后一个是竖着放置的小面包的方法数目，f2[i]表示不是竖着放置的小面包的数目。那么有递推式：

f1[i] = f1[i-1] + f2[i-1]

f2[i] = f1[i-2] + f2[i-2] + f1[i-1]

将初始化命名为
f1[1] = 1, f1[2] = 1;f2[1] = 0, f2[2] = 2;即可得到正确答案，想一想为什么…

讲上式简化可以得到f[i] = f[i-1] + 2 * f[i-2] 的递推式。
